## ✨ 核心特性 (Features)

本网站采用 **纯前端 (Vanilla HTML/CSS/JS)** 构建，未引入任何繁重的第三方 UI 框架，主打极致的加载速度与定制化交互：

*   **🤖 离线数字分身**：右下角悬浮智能助理，内置硬编码知识库[cite: 9]。零外部 API 请求，极速响应，完美解答关于我的实习、项目与技术栈等问题。
*   **🎨 Canvas 粒子特效与名片渲染**：全局动态粒子连线背景，并支持一键通过 Canvas 将个人信息实时渲染为精美 PNG 名片下载[cite: 9]。
*   **💻 Terminal 终端模拟**：首屏采用打字机终端 UI 渲染个人 JSON 档案，致敬极客文化[cite: 9]。
*   **📜 动态交互体验**：包含阅读进度条、3D 卡片倾斜悬浮、页面平滑滚动以及 Dark/Light 昼夜主题一键切换[cite: 9]。
