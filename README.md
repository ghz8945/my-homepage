# 郭洪志的个人主页 (Personal Portfolio)

这是一个基于纯前端技术栈（HTML/CSS/JS）开发的个人简历与展示主页。项目旨在清晰、直观地展示个人的专业技能、项目经历，并提供一个轻量级的互动体验。


## 💡 主要功能

* **个人档案展示**：结构化展示个人的教育背景、专业技能（如 Python、C++、Linux 等）及相关荣誉。
* **项目经验面板**：详细展示了“脑部肿瘤识别系统”和“RAG 智能客服系统”等核心项目背景与负责的具体技术细节。
* **交互式数字分身**：右下角内置一个基于原生 JavaScript 实现的离线聊天助手。该助手通过本地匹配逻辑，可快速回答关于本人履历、技能及联系方式的问题。
* **电子名片导出**：利用 HTML5 Canvas 技术，支持一键将个人核心信息渲染为名片图片并下载到本地。
* **响应式设计**：完全采用 CSS 原生媒体查询编写，适配 PC 端与移动端浏览。

## 🛠️ 技术栈

本项目未依赖任何第三方重量级框架，完全基于前端基础技术构建：
* **HTML5**：页面语义化结构
* **CSS3**：页面样式、CSS 动画与响应式布局
* **Vanilla JavaScript (ES6+)**：DOM 节点操作、Canvas 渲染以及离线聊天逻辑处理
